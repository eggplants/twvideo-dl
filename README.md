# twvideo-dl

A Simple CLI for downloading Videos from X, formally Twitter.

## Install

```bash
pip install twvideo-dl
```

## How to use

```shellsession
$ twvideo-dl https://twitter.com/YouTube/status/988453821851619330
Saved: 2023-12-19_02:41:39_1702921299_DbexH_IV0AEG3AA.mp4
```
